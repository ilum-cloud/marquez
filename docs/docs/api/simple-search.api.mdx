## Features

- **Fast Performance**: Optimized SQL queries with trigram indexes for efficient pattern matching
- **No External Dependencies**: Works without Elasticsearch/OpenSearch
- **Simple Results**: Returns essential metadata with references for UI navigation
- **Flexible Sorting**: Sort by name (alphabetical) or updated_at (newest first, default)
- **Type Filtering**: Filter results by datasets, jobs, or both
- **Namespace Filtering**: Restrict search to specific namespaces
- **Pattern Matching**: Supports prefix/postfix matching for flexible search

## Usage Examples

### Basic search
```bash
curl "http://localhost:8080/api/v1/search/simple?q=my-data"
```

### Search only datasets, sorted by name
```bash
curl "http://localhost:8080/api/v1/search/simple?q=analytics&filter=dataset&sort=name"
```

### Search within a specific namespace, limit results
```bash
curl "http://localhost:8080/api/v1/search/simple?q=pipeline&namespace=production&limit=10"
```
